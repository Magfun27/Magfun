Переменные в Python

Переменные в Python – это имена для хранения данных в памяти. Они служат контейнерами для значений: чисел, строк, списков и других объектов.

Объявление переменной

Пример объявления переменной и присваивания значения:

num = '68' # Создаем переменную num и присваиваем ей значение 68 print('значение', num) # Выводим значение

Многократное присвоение

Присвоение одного значения нескольким переменным:

num1 = num2 = num3 = 10 print(num1, num2, num3)
