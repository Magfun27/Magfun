Основы вывода данных в Python

Основные моменты: - Функция `print()`: Основная команда для вывода информации на экран. - Аргументы: Данные, помещённые внутрь круглых скобок функции print(), будут выведены на экран. - Строки: Текстовые значения, обозначаемые либо одинарными (`''`), либо двойными (`""`) кавычками. - Однострочный комментарий: Используется символ решётки (`#`). - Многострочный комментарий/docstring: Используются тройные кавычки (`''' '''` или `""" """`).

Примеры использования

print('Это моя первая программа')

Одинарные и двойные кавычки Можно свободно выбирать тип кавычек, главное — соблюдать правило: открывающая и закрывающая кавычка должны быть одинаковыми. Примеры:

print("This is a string with single quotes inside: 'Hello'") print('And this one has double quotes inside: "World"')

Эти конструкции полезны, когда внутри строки нужны разные типы кавычек. Важные правила: - Названия функций, включая print(), чувствительны к регистру символов. - Интерпретатор Python обрабатывает код построчно, выполняя команды последовательно.
